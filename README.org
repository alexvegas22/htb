#+TITLE: HTB 

*What is our app about*

We're making this app as a way to explore both Vue.js and Flask

*How do i run it?*

To run the entire application, execute the [[file:./deploy.sh][deploy]] script 
#+BEGIN_SRC BASH
./deploy.sh
#+END_SRC
If you only want to run the front end :
#+BEGIN_SRC BASH
cd htb-frontend
npm install
npm run dev
#+END_SRC
And for the backend :
#+BEGIN_SRC BASH
python -m venv .venv
pip install -r api/requirements.txt
. .venv/bin/activate
cd api/
flask run &
#+END_SRC
For the sock chat :
#+BEGIN_SRC BASH
cd ..
cd websocket/
npm install
node index.js &
#+END_SRC
